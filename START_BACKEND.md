# Backend Server Start Karne Ke Liye

## Method 1: Command Prompt Use Karein (Recommended)

1. **Command Prompt kholo** (PowerShell nahi)
2. **Server folder me jao:**
   ```
   cd "C:\Users\FBC\Desktop\food AI\oGjVnn\server"
   ```
3. **Server start karo:**
   ```
   node server.js
   ```

## Method 2: Batch File Use Karein

1. **File Explorer me jao:**
   ```
   C:\Users\FBC\Desktop\food AI\oGjVnn\server
   ```
2. **`start.bat` file par double click karo**

## Method 3: VS Code Terminal Me

1. **VS Code me new terminal kholo**
2. **Terminal type change karo: Command Prompt**
3. **Run karo:**
   ```
   cd server
   node server.js
   ```
